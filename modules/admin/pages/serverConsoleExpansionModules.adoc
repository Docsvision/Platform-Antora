= Блок "Модули расширения"

В разделе "Модули расширения" отображаются подразделы с настройками базовых и дополнительных модулей {dv}, для которых предусмотрена возможность настройки.

Модуль {pl} добавляет раздел настроек "Сервис полнотекстового поиска". В {cns} будет добавлен раздел настроек модуля _{dv} 5. Управление процессами_, если модуль установлен. Подробное описание приведено в документе модуль _{dv} 5. Управление процессами_. Руководство администратора, пункт _Настройки модуля в Консоли настройки {dv}_.
