= Подключить существующую базу данных

[IMPORTANT]
====
Не изменяйте конфигурацию сателлитных БД при подключении существующей БД {dv} 5.5!
====

. Откройте xref:serverConsoleDataBasesMaster.adoc[мастер БД {dv}].
+
.Мастер БД {dv}
image::DatabaseMaster.png[Мастер БД {dv}]
+
. Выберите режим *Подключить существующую базу данных, не представленную в списке*. Нажмите *Далее*.
. Укажите данные для подключения к СУБД. Нажмите *Далее*.
+
.Мастер БД {dv} в режиме подключения базы данных
image::DatabaseAttach_1.png[Мастер БД {dv} в режиме подключения базы данных]
+
include::dataBaseСreateNew.adoc[tags=dbSettings]
+
. Укажите параметры подключения БД и нажмите *Далее*.
+
.Мастер БД {dv} в режиме подключения базы данных
image::DatabaseAttach_2.png[Мастер БД {dv} в режиме подключения базы данных]
+
****
.. Укажите основные параметры БД:
+
include::dataBaseСreateNew.adoc[tags=name]
+
.. Нажмите кнопку *Настройки*, если требуется изменить конфигурацию xref:dataBaseSatellite.adoc[сателлитных баз данных] (только при подключении БД {dv} версии 5.4) и использования отдельных xref:dataBaseServiceTablesLocation[служебных таблиц].
+
При подключении БД {dv} 5.5 сохраняется существующая в данной БД конфигурация сателлитных БД и служебных таблиц.

image::updateDbOnAttach.png[Выбор модулей для обновления]

.Подтверждение параметров подключения БД

+
Если выбранная база данных была подключена или подключена сейчас к другому серверу {dv}, и лицензия {dv} не позволяет работать с одной БД одновременно нескольким серверам {dv}, при нажатии кнопки *Далее* появится диалоговое окно. В диалоговом окне будут предложены варианты отключения БД.

.Диалоговое окно "Внимание: параллельные серверы"
image::Database_Wizard_Parallel_Servers.png[Диалоговое окно "Внимание: параллельные серверы"]

Нажмите на кнопку *Продолжить*, чтобы система автоматически отключила БД от других серверов {dv}.

Данная процедура подразумевает, что БД будет xref:dataBaseMove.adoc[перемещена] (переключена) с одного сервера {dv} на другой.
****

****
Следующие шаги выполняются только в следующих случаях:

- Если требуется обновление БД.

- БД подключается из программы _{cns}_. При первоначальной настройке {dv} загрузка дополнительных модулей выполняется _Мастером настройки {dv}_.
****

[start=7]
. Дождитесь завершения загрузки SQL объектов.
+
. Выберите модули {dv}, настройки которых должны быть загружены в БД.
+
.Выбор модулей для загрузки настроек в БД
image::DatabaseCreate_5.png[Выбор модулей для загрузки настроек в БД]
+
Существующие в БД настройки загружаемых модулей будут перезаписаны.
+
. Дождитесь завершения загрузки настроек.
. Согласитесь с перезапуском сервисов {dv}, чтобы применить новые параметры.
+
.Запрос на перезапуск сервисов {dv}
image::configMasterReload.png[Запрос на перезапуск сервисов {dv}]
+
. Предоставьте сервисам {dv} xref:postConfigureServer.adoc#accountsSetup[необходимые права], в т.ч. права на новую БД.
