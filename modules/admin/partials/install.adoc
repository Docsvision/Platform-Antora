. Установите службу {of} следующей командой, предварительно обновив индекс пакетов:
+
[source,bash,subs=attributes]
----
sudo apt-get update
sudo apt-get install docsvision-{wt}
----
+
. Все настройки {of} хранятся в конфигурационном файле `appsettings.json`. Конфигурационный файл может быть изменён в любом текстовом редакторе, например `nano`.
