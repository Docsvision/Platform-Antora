= Работа с группами компонентов

== Создать группу компонентов

.Чтобы создать группу компонентов:
. Откройте _Справочник компонентов_.
. На панели инструментов нажмите кнопку image:buttons/add-components-group.png[Скобки с плюсом].
. В список групп будет добавлена новая группа, и открыто её описание.
. Выберите файл с иконкой для группы:
+
****
.. Нажмите кнопку *…* в поле `Иконка`.
.. Выберите файл ICO. Изображение иконки, а также путь к выбранному файлу отобразятся в поле `Иконка`.
****
+
. Введите название группы компонентов в поле `Название`.
+
****
Данное название участвует в формировании пути к файлам, загружаемым из _Справочника компонентов_ на компьютеры пользователя.
****
+
. Введите дополнительное описание для группы в поле `Описание`.
. Добавьте файлы в группу.
include::partial$save.adoc[]

== Удалить группу компонентов

. Откройте _Справочник компонентов_.
. Выберите группу компонентов в списке групп.
. На панели инструментов нажмите кнопку image:buttons/remove-components-group.png[Скобки с минусом], чтобы удалить группу компонентов из списка групп.
include::partial$save.adoc[]
. Компоненты, загруженные в удалённую группу, перестанут распространяться.
