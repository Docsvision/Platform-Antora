= Блок "Модули расширения"

В разделе "Модули расширения" отображаются подразделы с настройками базовых и дополнительных модулей {dv}, для которых предусмотрена возможность настройки.

Модуль {pl} добавляет раздел настроек "Сервис полнотекстового поиска". В _{cns}_ будет добавлен раздел настроек модуля _{wf}_, если модуль установлен. Подробное описание приведено в документации по администрированию модуля _{wf}_, раздел "xref:workflow:admin:system-settings.adoc[]".
